SECTION 3: THE 6TH METRIC - CHEMICAL PLAUSIBILITY
================================================================================

DEFINITION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Chemical Plausibility measures: "Does the discovered SAR make chemical sense?"

Metric Range: [0, 1]
  - 1.0: Highly plausible (expert would accept)
  - 0.8: Plausible with minor concerns
  - 0.5: Questionable (requires validation)
  - 0.2: Implausible (likely overfitting)
  - 0.0: Nonsensical (clear false positive)

COMPUTATION METHOD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Extract Important Features
-------
For model explanations, identify top-K atoms (K=5-10)
Extract structural context:
  - Functional groups they belong to
  - Position in molecule relative to known pharmacophores
  - Chemical properties (H-bond donor/acceptor, aromatic, etc.)

Example:
  Top atoms: {aromatic_C_para, halogen, carboxyl_O}
  Extract:
    - "Aromatic carbon in para-position to electron-withdrawing group"
    - "Halogen substituent on benzene ring"
    - "Carboxyl group as side chain"

Step 2: Create Prompt for Reasoning Model
--------
```
You are an expert medicinal chemist evaluating a machine learning model's 
SAR predictions. The model identified the following structural features as 
MOST IMPORTANT for [PROPERTY/TARGET]:

IDENTIFIED IMPORTANT FEATURES:
1. {Feature A description}
2. {Feature B description}
3. {Feature C description}
...

TASK:
Please evaluate the chemical plausibility of these features. For each feature:
1. Explain how/why it contributes to [PROPERTY/TARGET]
2. Identify relevant mechanistic principles
3. Note if it's consistent with known chemistry

Then provide:
- Overall Plausibility Score (0-10): How sensible is this SAR?
- Confidence Level: How confident are you in this assessment?
- Red Flags: Any concerns or potential artifacts?
- Suggestions: How would you experimentally validate this?

Think through carefully. Consider both mechanisms and common pitfalls.
```

Step 3: Query Reasoning Model
------
Send prompt to Claude/o1
Get response with explanation + score

Step 4: Extract Score
-----
Parse response for numerical score (0-10)
Normalize to [0, 1]: score_norm = score / 10
Handle edge cases (model refuses, unclear score, etc.)

PSEUDOCODE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

def compute_chemical_plausibility(model, explanations, smiles_list, 
                                 target_property, reasoning_api_key,
                                 model_name='claude-opus'):
    """
    Compute chemical plausibility of discovered SAR using reasoning model.
    
    Args:
        model: Trained ML model
        explanations: Atom importance scores (n_samples, n_atoms)
        smiles_list: Test SMILES strings
        target_property: str, e.g., "BBB penetration" or "IC50 potency"
        reasoning_api_key: API key for reasoning model
        model_name: 'claude-opus', 'o1', etc.
    
    Returns:
        plausibility_score: float [0, 1]
        reasoning: str (explanation from model)
        scores_per_feature: dict (score for each identified feature)
    """
    
    # Step 1: Extract top important atoms
    top_k = 8
    top_indices = np.argsort(-np.mean(np.abs(explanations), axis=0))[:top_k]
    
    # Step 2: Get structural descriptions
    features = extract_structural_context(smiles_list, top_indices)
    
    # Step 3: Build prompt
    prompt = build_plausibility_prompt(features, target_property)
    
    # Step 4: Query reasoning model
    response = query_reasoning_model(
        prompt=prompt,
        api_key=reasoning_api_key,
        model=model_name
    )
    
    # Step 5: Parse response
    plausibility_score = extract_score(response)  # 0-10 → 0-1
    reasoning = extract_explanation(response)
    feature_scores = extract_per_feature_scores(response)
    
    return plausibility_score, reasoning, feature_scores

"""
Wrapper for reasoning model APIs (Claude, o1, etc.)
Handles prompting, response parsing, error handling.
"""

import anthropic
import openai
import re
from typing import Tuple, Dict, Any
import json

class ReasoningModelWrapper:
    """
    Unified interface for reasoning models.
    Supports: Claude-Opus, o1, o1-preview
    """
    
    def __init__(self, model_name='claude-opus', api_key=None):
        self.model_name = model_name
        
        if 'claude' in model_name:
            self.client = anthropic.Anthropic(api_key=api_key)
            self.model_type = 'claude'
        elif 'o1' in model_name:
            self.client = openai.OpenAI(api_key=api_key)
            self.model_type = 'o1'
        else:
            raise ValueError(f"Unknown model: {model_name}")
    
    def query(self, prompt: str, max_tokens: int = 2000) -> str:
        """
        Send prompt to reasoning model, get response.
        
        Args:
            prompt: str, chemistry prompt
            max_tokens: max response length
        
        Returns:
            response: str, model's response
        """
        
        if self.model_type == 'claude':
            response = self.client.messages.create(
                model=self.model_name,
                max_tokens=max_tokens,
                thinking={
                    "type": "enabled",
                    "budget_tokens": min(10000, max_tokens)
                },
                messages=[{"role": "user", "content": prompt}]
            )
            return response.content[0].text
        
        elif self.model_type == 'o1':
            response = self.client.chat.completions.create(
                model=self.model_name,
                messages=[{"role": "user", "content": prompt}]
            )
            return response.choices[0].message.content
    
    def extract_score(self, text: str) -> float:
        """Extract numerical score (0-10) from response."""
        
        # Try explicit mention
        patterns = [
            r'(?:score|rating|plausibility).*?(\d+)\s*(?:/10|out of 10)',
            r'(\d+)\s*(?:/10|out of 10)',
            r'score\s*[:\s]*(\d+)',
        ]
        
        for pattern in patterns:
            match = re.search(pattern, text, re.IGNORECASE)
            if match:
                score = int(match.group(1))
                return min(10, max(0, score)) / 10.0  # Normalize to [0, 1]
        
        # No explicit score - estimate from text
        if 'highly plausible' in text.lower() or 'excellent' in text.lower():
            return 0.9
        elif 'plausible' in text.lower() or 'reasonable' in text.lower():
            return 0.75
        elif 'questionable' in text.lower() or 'concern' in text.lower():
            return 0.5
        elif 'implausible' in text.lower() or 'unlikely' in text.lower():
            return 0.2
        else:
            return 0.5  # Default to neutral
    
    def extract_explanation(self, text: str) -> str:
        """Extract key explanation from response."""
        lines = text.split('\n')
        # Return first 5-10 lines as summary
        return '\n'.join(lines[:10])

# Example usage
if __name__ == "__main__":
    wrapper = ReasoningModelWrapper(
        model_name='claude-opus',
        api_key='your-api-key'
    )
    
    prompt = "Is this SAR plausible?..."
    response = wrapper.query(prompt)
    score = wrapper.extract_score(response)
    print(f"Score: {score}")

                if mol is None or idx >= mol.GetNumAtoms():
                    continue
                
                atom = mol.GetAtomWithIdx(idx)
                
                # Feature description
                feature_desc = {
                    'atom_symbol': atom.GetSymbol(),
                    'is_aromatic': atom.GetIsAromatic(),
                    'hybridization': str(atom.GetHybridization()),
                    'formal_charge': atom.GetFormalCharge(),
                    'neighbor_types': [n.GetSymbol() for n in atom.GetNeighbors()]
                }
                features.append(feature_desc)
                break
        
        return features
    
    def build_prompt(self, features, target_property):
        """Build chemistry prompt for reasoning model."""
        
        feature_text = "IDENTIFIED IMPORTANT FEATURES:\n"
        for i, feat in enumerate(features, 1):
            feature_text += f"{i}. {feat}\n"
        
        prompt = f"""You are an expert medicinal chemist.

A machine learning model identified these structural features as MOST 
IMPORTANT for {target_property}:

{feature_text}

TASK:
1. For each feature, explain: Why does this promote {target_property}?
2. Does this SAR make chemical sense? (0-10 score)
3. Any red flags or concerns?

Think carefully through chemical principles.
"""
        return prompt
    
    def compute(self, model, explanations, smiles_list, target_property):
        """
        Compute chemical plausibility score.
        
        Returns: (score, reasoning)
        """
        
        # Step 1: Extract top features
        top_k = 8
        top_indices = np.argsort(-np.mean(np.abs(explanations), axis=0))[:top_k]
        
        # Step 2: Get descriptions
        features = self.extract_structural_features(smiles_list, top_indices)
        
        # Step 3: Build prompt
        prompt = self.build_prompt(features, target_property)
        
        # Step 4: Query reasoning model
        response = self.reasoner.query(prompt)
        
        # Step 5: Extract score
        score = self.reasoner.extract_score(response)
        reasoning = self.reasoner.extract_explanation(response)
        
        return score, reasoning

    stability, 
    consistency, 
    faithfulness,

    plausibility=None
):
    """
    Compute Chemical Awareness Score with optional 6th metric.
    
    If plausibility is None, use 5-metric version (original).
    If plausibility is provided, use 6-metric version (with reasoning model).
    """
    
    if plausibility is None:
        # Original 5-metric CAS
        cas = (0.25 * stability + 
               0.20 * consistency + 
               0.25 * faithfulness +

        cas = (0.22 * stability + 
               0.18 * consistency +

               0.13 * parsimony +

    return cas

RISK 1: Reasoning model is unreliable
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Risk: Reasoning model hallucinates or gives low-quality chemistry advice

Mitigation:
  ✓ Always include manual review of top-3 models' plausibility explanations
  ✓ For paper: Show actual reasoning text (not just scores) in supplementary
  ✓ Cross-validate: High CAS should correlate with high plausibility
  ✓ Test on benchmarks where we know answer (BBBP, etc.)
  ✓ If reasoning is off, it should correlate inversely with SAR recovery

---

RISK 2: Cost/latency too high
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Risk: API queries too slow or expensive to finish paper on time

Mitigation:
  ✓ Start with small subset (10 molecules) to validate approach
  ✓ Use Claude-Opus (cheaper, faster) before switching to o1
  ✓ Can skip reasoning for full model selection, only validate top-3 models
  ✓ Budget: ~$25-50 for full paper (very affordable)
  ✓ Time: ~1 hour total for all queries

---

RISK 3: Reasoning metric doesn't improve results
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Risk: Plausibility scores don't meaningfully change CAS

Mitigation:
  ✓ Lower weight to 12% (not dominant)
  ✓ Use as validation, not primary selection
  ✓ If doesn't help, just document in paper: "reasoning adds confidence 
    but not primary signal"
  ✓ Still novel: first paper to add reasoning validation

---

RISK 4: Reasoning model contradicts ground truth
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Risk: Reasoning model says SAR is implausible, but we know it's true

Mitigation:
  ✓ Test on benchmarks; validate correlations
  ✓ If mismatch, improve prompts or switch to o1 (better reasoning)
  ✓ Document discrepancies: "Reveals limitations of reasoning models"
